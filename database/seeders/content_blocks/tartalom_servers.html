Szerver alkalmazások az unRAID-en

Az unRAID ideális platform különböző szerver alkalmazások futtatására Docker konténerekben. Media streaming-től a fájlmegosztásig, minden egy helyen, egyszerűen kezelhető webes felületen keresztül.

Media Server - Plex, Jellyfin, Emby

Professzionális média streaming megoldások, amelyek lehetővé teszik a film-, zene- és fotógyűjtemények rendszerezését és streamelését bármilyen eszközre. A Plex a legnépszerűbb választás gazdag funkcionalitással és széles eszköztámogatással.

Plex Media Server: Automatikus metadata letöltés, transzkódolás, távoli elérés, mobil alkalmazások. Támogatja a hardveres gyorsítást (GPU transcoding) és a multi-user környezetet.

Jellyfin: Nyílt forráskódú alternatíva, teljesen ingyenes, nincs előfizetési díj. Hasonló funkciók mint a Plex, de teljes adatkontrollal.

Kodi: Média center alkalmazás, amely közvetlenül az unRAID-en vagy kliens eszközökön futtatható. Testreszabható felület, kiegészítők (add-ons) támogatása.

File Server - Nextcloud, Syncthing

Saját felhő tárolás és fájlszinkronizálás, amely teljes adatkontrollt biztosít. Alternatíva a Google Drive, Dropbox és OneDrive szolgáltatásokhoz, de minden adat a saját szervereden marad.

Nextcloud: Teljes körű felhő platform fájlmegosztással, naptárral, névjegyekkel, dokumentumszerkesztéssel. Webes felület, mobil és desktop alkalmazások. Támogatja a titkosítást és a kétfaktoros hitelesítést.

Syncthing: Peer-to-peer fájlszinkronizálás több eszköz között. Nincs központi szerver, titkosított kapcsolat, automatikus szinkronizálás.

SMB/NFS megosztások: Natív fájlmegosztás Windows, macOS és Linux rendszerekhez közvetlenül az unRAID-ből.

Download Server - qBittorrent, SABnzbd, Transmission

Automatizált letöltési megoldások torrent és Usenet protokollokhoz, amelyek 24/7 futhatnak alacsony energiafogyasztással. Webes felületen keresztül kezelhetők bárhonnan.

qBittorrent: Nyílt forráskódú torrent kliens webes felülettel. VPN integráció, automatikus kategorizálás, RSS feed támogatás.

SABnzbd: Usenet letöltő automatikus kicsomagolással és javítással. Integrálható média automatizálási eszközökkel (Sonarr, Radarr).

Sonarr/Radarr: Automatikus média menedzsment TV sorozatokhoz és filmekhez. Automatikus letöltés, átnevezés, könyvtárkezelés.

Home Automation - Home Assistant, Node-RED

Okosotthon központ, amely összeköti és automatizálja a különböző IoT eszközöket. Helyi kontroll, nincs felhő függőség, teljes adatvédelem.

Home Assistant: Több mint 2000 integráció (Philips Hue, Sonoff, Xiaomi, stb.). Automatizálások, jelenetek, dashboard-ok. Mobil alkalmazás iOS-re és Androidra.

Node-RED: Vizuális programozási eszköz IoT és automatizálási folyamatokhoz. Drag-and-drop felület, széles körű node támogatás.

Mosquitto MQTT: Üzenetküldő broker IoT eszközök közötti kommunikációhoz. Könnyű, gyors, megbízható.

Web Server & Development - Nginx, GitLab, Portainer

Webszerver és fejlesztői eszközök saját projektek hostolásához és kezeléséhez. Tesztkörnyezetek, verziókezelés, CI/CD pipeline-ok.

Nginx Proxy Manager: Reverse proxy SSL tanúsítványokkal (Let''s Encrypt). Egyszerű webes felület, automatikus SSL megújítás, domain routing.

GitLab: Teljes DevOps platform Git verziókezeléssel, CI/CD, issue tracking, wiki. Saját Git szerver teljes kontrollal.

Portainer: Docker konténer menedzsment grafikus felülettel. Könnyű konténer kezelés, stack deployment, monitoring.

Database & Monitoring - MariaDB, PostgreSQL, Grafana

Adatbázis szerverek és monitoring eszközök alkalmazások backend-jéhez és rendszer felügyeletéhez. Teljesítmény nyomon követés, metrikák, riasztások.

MariaDB/MySQL: Relációs adatbázis szerver webalkalmazásokhoz. Gyors, megbízható, széles körben támogatott.

PostgreSQL: Fejlett nyílt forráskódú adatbázis komplex lekérdezésekhez és nagy adathalmazokhoz.

Grafana + Prometheus: Monitoring és vizualizációs platform. Szerver metrikák, Docker statisztikák, egyedi dashboard-ok, riasztások.
