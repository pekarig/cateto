Hálózat és jogosultságkezelés

Az unRAID fejlett hálózati és hozzáférés-kezelési lehetőségeket kínál, amely lehetővé teszi a biztonságos és rugalmas hálózati konfigurációt.

Megosztás

Fájlmegosztási protokollok

Több fájlmegosztási protokoll támogatása, amely lehetővé teszi a különböző operációs rendszerek közötti zökkenőmentes együttműködést.

  • SMB - Windows megosztások
  • NFS - Linux/Unix megosztások
  • AFP - macOS megosztások
  • FTP/SFTP támogatás

Jogosultság

Felhasználó- és jogosultságkezelés

Részletes felhasználói és jogosultsági rendszer, amely lehetővé teszi a hozzáférés pontos szabályozását.

  • Felhasználói fiókok kezelése
  • Csoportok és jogosultságok
  • Megosztás szintű hozzáférés
  • Csak olvasható/írható jogok

Elkülönítés

Docker és VM hálózatok

Elkülönített hálózati környezetek Docker konténerek és virtuális gépek számára, amely növeli a biztonságot és rugalmasságot.

  • Docker egyedi hálózatok
  • VM dedikált hálózati interfészek
  • Hálózati izoláció
  • Port forwarding
