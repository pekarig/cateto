Szoftverek és alkalmazások az unRAID platformon

Az unRAID beépített Docker kezelőfelülettel és Community Applications katalógussal rendelkezik, amely lehetővé teszi számos alkalmazás egyszerű telepítését és kezelését, beleértve az AI-alapú megoldásokat is.

Docker

Docker és alkalmazások

Az unRAID beépített Docker kezelőfelülettel rendelkezik, valamint Community Applications katalógussal, amely egyszerűvé teszi a konténerizált alkalmazások telepítését.

  • Plex, Jellyfin, Nextcloud
  • Home Assistant
  • GitLab, MariaDB, Nginx

AI Stack

AI alkalmazások

GPU passthrough és NVIDIA Driver plugin segítségével CUDA gyorsítás is elérhető, amely lehetővé teszi AI modellek hatékony futtatását.

  • Ollama - lokális LLM futtatás
  • Stable Diffusion - képgenerálás
  • Whisper - speech-to-text
  • TensorFlow, PyTorch

GPU

GPU passthrough és gyorsítás

Az NVIDIA GPU-k teljes potenciáljának kihasználása Docker konténerekben és virtuális gépekben, CUDA gyorsítással AI és compute feladatokhoz.

  • NVIDIA Driver plugin
  • CUDA Container Toolkit
  • GPU passthrough VM-ekhez
  • AI modellek gyorsítása
