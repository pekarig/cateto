Virtuális gépek az unRAID-en

KVM/QEMU alapú virtualizáció érhető el grafikus webes kezelőfelülettel, amely lehetővé teszi különböző operációs rendszerek futtatását natív teljesítménnyel.

Platform

KVM/QEMU virtualizáció

Linux alapú, natív virtualizációs megoldás nagy teljesítménnyel, amely lehetővé teszi különböző operációs rendszerek futtatását.

  • Natív Linux virtualizáció
  • Hardver gyorsítás támogatás
  • Grafikus webes kezelőfelület
  • Egyszerű VM létrehozás

OS

Támogatott operációs rendszerek

Széles körű operációs rendszer támogatás, beleértve a Windows, Linux és BSD rendszereket is.

  • Windows VM-ek (7, 10, 11, Server)
  • Linux disztribúciók (Ubuntu, Debian, CentOS)
  • BSD rendszerek (FreeBSD, OpenBSD)
  • macOS (Mackintosh)

Passthrough

PCIe passthrough

Fizikai hardver eszközök közvetlen hozzárendelése virtuális gépekhez, amely natív teljesítményt biztosít.

  • GPU passthrough (videokártyák)
  • HBA passthrough (tárolóvezérlők)
  • USB passthrough (perifériák)
  • Hálózati kártyák

Teljesítmény

VirtIO optimalizáció

Optimalizált virtuális eszközmeghajtók, amelyek jelentősen javítják a VM teljesítményét és hatékonyságát.

  • VirtIO lemez meghajtók
  • VirtIO hálózati interfészek
  • Jobb I/O teljesítmény
  • Alacsonyabb CPU használat
